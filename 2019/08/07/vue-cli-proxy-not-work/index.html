也就是说需要进行跨域访问的api接口域名应该跟vue-cli本地启动的服务器域名保持一致，当服务器发现能匹配到静态资源的时候就内部消化，返回静态资源，而当请求不到的时候才会换用配置的代理域名进行转发访问。
