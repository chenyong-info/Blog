小米Note尾插更换图解

发表于 2017-07-21 | 分类于 数码捣腾 | 阅读次数


  作为小米跳水队先锋选手的第一代Note，在同价位情况下无论设计感还是做工、体验确实都好过红米系列一个档次，无奈脑残的方形无防呆USB充电口这个猪队友设计师把一切都给毁了，网上搜“小米方形充电口”各种吐槽，多数是半到一年左右就开始出现各种充电问题，根源就在于在没分清楚方向反插的情况下日积月累导致充电口接触簧片损坏。这时候在保还好说、不在保一修就是85￥的材料费+40￥的人工费，这尼玛手机也才几百，修个充电口就100多，最后决定自己上。

  维修方案大致有2种，一种是只替换MicroUSB充电头，优点是价格低廉，运费比材料贵系列，缺点是技术难度稍高，还需要焊接技能加持；另一种是整体替换尾插板，材料成本10-20多不等，优点是无需焊接，仅需拆机插拔替换即可，相对简单。


  在购买前首先要确定自己手机的版本，标准版还是顶配版（尾插排线会有所不同），这里选用的是带拆机工具8件套的标准版尾插——25.97￥，不需要拆机工具的话在15￥左右可以搞定。


  首先关机，用取卡针取出卡槽。


  接着用吸盘吸住背壳一角，用力上拉就可以拉开一条缝隙。


  拨片插入缝隙中慢慢的翘开各处卡扣，最后就能把背壳取下来。


  背壳拆下后将上图中圈出的螺丝孔位全部拆除（上9下8），取下两块盖板，因为螺丝规格有差别，注意记录好不同螺丝的位置以便后续装回。


  拿出尾插配件对比一下，要替换的位置已经一目了然了。接下来就是拆解电池，首先是把左上角的排线拔下，接着揭开电池下发2块黑条，其下连接的是3M无痕双面胶，一直往外拉，直到完全拉出，电池就可以自然脱落了。


  电池取下后可以看到尾插的全貌，首先拔下上方的排线插头，然后依次取下板上的电池和天线插头，接着从左下开始逐步敲开尾插板，因为是整体替换，所以不用担心配件的损坏问题，其是通过双面胶贴合在下发金属上，并没有螺丝的固定。这里需要注意的地方就是左下全出的地方有一块黑色压片熟料小板，也是贴合在尾插线头上的，需要取下来，不要当废料扔掉了。

  后续的过程就是替换过后逆向还原了，只要做好笔记并没有太大难度，电池底部的双面胶在还原时加或不加关系都不太大。

  最后如果看图还无法明白的地方可以参考这个完整版的小米Note拆机视频
